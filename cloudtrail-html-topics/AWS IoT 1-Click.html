Learn about AWS IoT 1-Click

When did AWS IoT 1-Click reach end-of-life (EOL)?

AWS discontinued AWS IoT 1-Click on December 16, 2024. As part of this, AWS IoT 1-Click transitioned into its EOL in the following phases.

  • On December 15, 2023, AWS announced our plan to discontinue AWS IoT 1-Click, and stopped onboarding new customers.

  • On May 16, 2024, AWS discontinued access to the AWS IoT 1-Click management console.

  • On December 16, 2024, AWS IoT 1-Click reached its end-of-life. AWS stopped all support for the service. The AWS IoT 1-Click APIs, AWS CLI, and mobile applications were shut down.

What happened to my AWS IoT 1-Click projects and buttons after the end-of-life date?

As of December 16, 2024, calls to the AWS IoT 1-Click APIs no longer work, and AWS IoT 1-Click buttons will no longer trigger associated AWS Lambda functions using AWS IoT Core. Instead, you can build alternative solutions with AWS IoT Core. For more information, see Getting started with AWS IoT Core .

Will my AWS IoT 1-Click buttons work after the end-of-life date?

No. AWS IoT 1-Click buttons won't be able to connect to AWS IoT Core or trigger Lambda functions.

Will I be billed for my remaining AWS IoT 1-Click buttons after the end-of-life date?

No. AWS will not bill you for AWS IoT 1-Click on or after December 16, 2024.

Do I need to remove AWS IoT 1-Click references from my AWS CloudFormation templates?

Yes. You must remove all AWS IoT 1-Click references from your AWS CloudFormation templates. Not doing so will cause your templates to fail.

Who do I contact for questions regarding AWS IoT 1-Click?

For any questions, contact AWS Support .
